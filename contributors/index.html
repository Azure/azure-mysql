Get connected with the Azure Database for MySQL Contributor program

The Azure Database for MySQL Contributor program recognizes Azure Database for MySQL community members who actively support one another by sharing Azure MySQL-related content, announcements, and product news via social media channels.

Azure Database for MySQL Contributors range in expertise and engagement level, but they all share one common goal—to help others get connected and expand their knowledge of Azure Database for MySQL. This would include MySQL community members, Azure Database for MySQL users and customers, MySQL influencers, Microsoft MVPs and partners, Microsoft internal members (GBBs, CSAs, CSS team members, Solution Specialists, Sellers, Consultants and FastTrack team engineers who help with migrations, etc.). In addition to recognizing users who regularly create and post content, this program encourages new users to get involved in community discussions.

Why should I become an Azure Database for MySQL Contributor?

If you’re passionate about Azure Database for MySQL and helping your peers learn and do more with the service, this program is for you. The Azure Database for MySQL Contributor program brings together like-minded database professionals through social media to create a community of people who help each other solve challenges, get connected with a global network of users, and achieve more through Azure Database for MySQL. Azure Database for MySQL Contributor members also have opportunities to collaborate directly with the Azure Database for MySQL engineering team to create new content and highlight their expertise on specific topics.




What are the benefits?

  • Official recognition for your support of the Azure Database for MySQL community.
  • An exclusive badge for each membership tier.
  • Azure Database for MySQL Contributor spotlights and visibility through official Azure MySQL channels.
  • Exclusive access to the Azure MySQL engineering team, including events and Q&As.
  • Content collaboration opportunities with other Azure Database for MySQL Contributor members and the Azure MySQL team.


What does it take to be an Azure Database for MySQL Contributor?

Azure Database for MySQL advocates of all levels are welcome to join the Azure Database for MySQL Contributor Program. If you’re interested in helping to build an active community by engaging with and producing Azure MySQL-related content, we encourage you to apply.

Membership levels:

All Level I activities and one of the following activities: All Level II activities and one of the following activities:
Tier Membership Requirements
Level I - Gold Contributor
  • Follow us on X (Twitter), LinkedIn, and YouTube
  • Amplify @AzureDBMySQL posts on X (Twitter) or #AzureDBMySQL posts on LinkedIn (4 shares/month - Each share must include a Like and a Quoted Retweet with original comment to qualify)
Level II - Emerald Contributor
  • Partner with us on developing a new Azure Database for MySQL official blog post / code sample / reference architecture (1 collateral / quarter).
  • Post new Azure Database for MySQL specific content on social media, personal blog or other online forums. (1 post / month).
Level III - Platinum Contributor
  • Amplify an article from our MS Learn documentation OR an Azure Database for MySQL team blog post and/or Azure Database for MySQL channel YouTube video (2 posts/month - Each post must include an original comment to qualify).
  • Star in an Azure Database for MySQL official video or show on YouTube (1 video or show / quarter).
  • Create or drive Azure Database for MySQL User Groups or Communities, either on social media or in-person local communities. (1 engagement / quarter).

How do I apply?

Becoming an Azure Database for MySQL Contributor is easy! Simply select sign-up link below and fill out the form by specifying your X (Twitter) or other social handle and specify the membership level you're committing to, and then start (or continue) engaging. Share Azure MySQL-related posts and content using the hashtag #MySQLContributor. After you’ve signed up, you’ll be able to collect your exclusive badge and add it to your social media account.

FAQs

  1. How does the Azure Database for MySQL Contributor program differ from the Microsoft MVP program?

    Anyone who is interested in getting connected with the Azure Database for MySQL community is encouraged to join the program and begin sharing Azure Database for MySQL-related content. Community members, users, and advocates of all experience levels are eligible to become Azure Database for MySQL Contributor program members. The only program requirements are that members meet the minimum engagement levels and be interested in helping others get connected and expand their knowledge of Azure Database for MySQL.

  2. When does my Azure Database for MySQL membership expire?

    Azure Database for MySQL membership is valid for as long as you choose to engage with the program. If you no longer wish to participate in the program, please contact us at azuremysqlcontribute@microsoft.com to let us know. Each year, members are required to recommit to receive an updated badge.

  3. Can I still become a member of the Azure Database for MySQL Contributor program if I don't have an X (Twitter) account?

    Yes! If you're sharing and engaging with Azure Database for MySQL-related content and want to connect with other community members, we encourage you to join the program. (Please share your public-facing social handle and platform in the Twitter handle form field.)

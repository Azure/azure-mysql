Azure Database for MySQL Contributor program


The Azure Database for MySQL Contributor program is designed to recognize members of the Azure Database for MySQL community who actively collaborate to share Azure MySQL-related content, announcements, and product news via social media channels.

Ranging in expertise and engagement level, MySQL Contributor program participants include:

  • MySQL community members and influencers
  • Azure Database for MySQL users and customers
  • Microsoft MVPs and partners
  • Microsoft internal team members

Despite this breadth of experience, all Azure MySQL contributors have a common goal — to help others connect and expand their knowledge of Azure Database for MySQL.






Why should I join the Azure Database for MySQL Contributor program?

Because you're passionate about Azure Database for MySQL and helping others to learn and do more with the service, that’s why! As a program member, you’ll join efforts with other, like-minded database professionals to create valuable content and video assets and amplify them via a variety of social media channels.


What are the benefits of joining the program?

  • Official recognition for your support of the Azure Database for MySQL community.
  • An exclusive badge for each membership tier.
  • Azure Database for MySQL Contributor spotlights / visibility via official Azure MySQL channels.
  • Opportunities to collaborate directly with the Azure Database for MySQL engineering team to develop collateral highlighting your areas of expertise.





What is expected of the program participants?

From sharing our social media posts to appearing in a Live Webinar, there are a variety of levels you can sign up for, as described in the following table.

Participation Level Requirements
Level I - Gold Contributor

Complete both of the following activities:

  • Follow us on X (Twitter), LinkedIn, and YouTube.
  • Amplify posts from @AzureDBMySQL on X (Twitter) or Azure Database for MySQL on LinkedIn (4 shares/month - Each share must include a Like and a Quoted Retweet with original comment to qualify).
Level II - Emerald Contributor

All Level I activities and one of the following activities:

  • Amplify an article from our Microsoft Learn documentation, a blog post from Azure Database for MySQL Blog or a video from the Azure Database for MySQL official YouTube channel (2 posts/month - Each post must include an original comment to qualify).
  • Post new Azure Database for MySQL specific content on social media, personal blog or other online forums. (1 post / month).
Level III - Platinum Contributor

All Level II activities and one of the following activities:

  • Partner with us on developing a new Azure Database for MySQL official blog post / code sample / reference architecture (1 collateral / quarter).
  • Star in an Azure Database for MySQL official video or show on YouTube (1 video or show / quarter).
  • Create / drive Azure Database for MySQL user groups or communities, on social media or via in-person local communities. (1 engagement / quarter).



How do I apply?

To join the Azure Database for MySQL Contributor program, follow the following simple steps!

  • Select the Sign up button below.
  • In the form that opens, provide your X (Twitter), LinkedIn or other social handle and then select the participation level you are committing to.
  • After signing up, go ahead and start (or continue) engaging! Share Azure MySQL-related posts and content, and don't forget to use the hashtag #AzureDBforMySQL.
